核心觀念框架

Concept 01
成長型思維

能力不是固定的，而是透過努力與策略可以發展的。把挫折看成學習，把挑戰看成機會。

Concept 02
刻意練習

進步來自在舒適圈邊緣的有目的練習，而非單純的重複。找到弱點，專注突破，獲得即時回饋。

Concept 03
目標與系統

目標告訴你方向，系統決定你進步的速度。兩者缺一不可，但每日執行的是系統，不是目標。

Concept 04
覆盤迭代

定期回顧是成長的加速器。沒有覆盤的努力只是重複，有覆盤的努力才是螺旋上升。

「你無法連接向前看的點，只能回頭才能連接它們。所以你必須相信這些點在未來某種程度上會連在一起。」
— Steve Jobs, Stanford Commencement Address, 2005

L01
個人成長地圖與證照規劃
Growth Roadmap · 40 min
展開 ›

// 單元說明

成長不應該是隨機的，而是有方向感的積累。個人成長地圖是一份描繪「你想成為誰」與「你目前在哪裡」之間路徑的文件，涵蓋知識、技能、證照、人際關係與人生經驗各個面向。

本單元帶你繪製自己的成長地圖，以你目前的三張證照為錨點（社工師、健康管理師、顧客服務管理），思考它們如何組合成一個獨特的專業定位，以及未來 1–3 年你想繼續累積的方向。

// 本節重點

  • 個人成長地圖的三個維度：知識深度 / 技能廣度 / 人生經驗
  • 以證照為錨點的專業定位分析
  • 社工 × 健康管理 × 顧客服務的跨域整合可能性
  • 設計未來 3 年的證照與學習路徑
  • 區分「想要」的成長與「需要」的成長
本節作業 畫出你的個人成長地圖：列出已有的知識、技能、證照，以及未來 3 年你想增加的能力。思考你的「跨域組合」有什麼獨特之處，寫下一段 100 字的個人專業定位描述。 作業時間：1.5 小時
L02
顧客服務管理能力認證
Customer Service Cert · 進行中 3%
展開 ›

// 課程概述

初階顧客服務管理專業能力認證由 IBVE 國際商業職業教育協會開設，獲 ISCA、ACE、NASM、AFAA 等國際機構認可，總計 10 CEUs 學習時數，採線上隨時上線的彈性學習模式。

顧客服務能力是跨行業通用的核心軟實力，尤其在社工與健康管理的服務場景中，理解客戶（案主）心理、建立信任關係、處理情緒衝突的能力，將大幅提升你的專業服務品質與影響力。

// 學習重點與應用

  • 顧客情緒管理：如何在高壓服務場景中保持穩定
  • 積極傾聽技術：讓對方感到被理解的溝通方式
  • 投訴處理流程：從衝突到解決的標準化步驟
  • 與社工實務的連結：案主服務中的關係建立技巧
  • 學習建議：每週安排 1–2 次課程，每次 30–45 分鐘
本週任務 本週完成 IBVE 課程平台上至少 2 個單元，並在學習後寫下：這個概念如何應用到你目前的工作或生活中？把學習紀錄加入你的知識庫筆記。 目標進度：3% → 15%
L03
健康管理師認證課程
HMCA Health Manager Cert · 進行中 34%
展開 ›

// 課程概述

HMCA 國際 C 級健康管理師認證課程由 HMCA 國際健康管理照護協會研發，結合健康科學、營養、心理、預防醫學等多面向知識，目標是培訓具備實務應用能力的健康管理師。目前進度 34%，距離完成仍有一段路，但已建立相當的基礎知識。

健康管理師的知識體系與社工師的服務工作高度互補——尤其在老人服務、身心障礙服務、社區健康促進等場域，擁有健康管理的專業視角，能夠為案主提供更全面的服務方案。

// 剩餘學習重點

  • 已完成（約 34%）：健康管理基礎概念與健康評估
  • 待完成：營養管理與飲食介入方法
  • 待完成：運動處方設計與身體活動指導
  • 待完成：慢性病預防與健康促進策略
  • 待完成：心理健康與壓力管理介入
本週任務 本週在 HMCA 課程平台上推進至少 3 個單元，目標把進度從 34% 推進到 40%。學習後用費曼筆記法為一個核心概念寫下「用最簡單語言解釋給非專業者聽」的摘要。 目標進度：34% → 40%
L04
OKR 目標設定與季度規劃
OKR Planning · 45 min
展開 ›

// 單元說明

OKR（Objectives and Key Results）是由 Intel 和 Google 等科技公司廣泛採用的目標管理框架，但它同樣適用於個人成長——用明確的「目標」定義你想要什麼，用可衡量的「關鍵結果」確認你是否真的在進步。

個人 OKR 的運作週期建議以「季」為單位：每季初設定 2–3 個目標，每個目標搭配 2–3 個具體、可衡量的關鍵結果。季末評分並覆盤，作為下季設定的參考。這個框架能幫你把長期夢想拆解成每季可執行的具體里程碑。

// 本節重點

  • OKR 的結構：Objective（質性方向）+ Key Results（量性指標）
  • 好的 OKR 特徵：激勵性、明確性、可量化、有時限
  • 個人 OKR 範例設計（以本課程為情境）
  • OKR vs. SMART 目標：兩種框架的互補使用方式
  • 季度中期檢查點：如何在 OKR 偏軌時即時校正
本節作業 為本季度（Q1）設定你的個人 OKR：寫下 2 個目標，每個目標各設定 2–3 個可量化的關鍵結果。其中一個目標建議與你目前進行中的認證課程相關，另一個可涵蓋個人生活或財務面向。 作業時間：1 小時
L05
心態建立：成長型思維
Growth Mindset · 40 min
展開 ›

// 單元說明

Carol Dweck 在《心態致勝》中提出：人對自身能力的信念分為「固定型思維」與「成長型思維」。固定型思維者認為能力是天生固定的，因此害怕挑戰、逃避失敗；成長型思維者相信能力可以透過努力發展，因此把困難視為機會，把失敗視為資訊。

成長型思維不是喊口號，而是一套可以練習的思維習慣。本單元帶你識別自己在哪些領域容易陷入固定型思維，並透過具體的認知重構練習，逐步培養更有韌性的成長心態。

// 本節重點

  • 固定型思維 vs. 成長型思維的具體行為差異
  • 識別你的固定型思維觸發情境
  • 「尚未」的力量：把「我不會」改成「我還不會」
  • 刻意練習的四個要素：目標 / 專注 / 回饋 / 突破舒適圈
  • 失敗日誌：用記錄失敗來提取學習，而非逃避
本節作業 列出三個你過去曾用「固定型思維」看待的能力或情境（例如「我天生不擅長記東西」）。用成長型思維重新框架這三句話，改成「我目前還沒有找到適合我的記憶方法」。接著為其中一項寫下一個可以立即嘗試的小行動。 作業時間：30 分鐘
L06
年度覆盤與人生作業系統整合
Annual Review · 50 min
展開 ›

// 單元說明

這是整個課程的最終章，也是最重要的一章。你已經學習了七個模組的管理方法，但學習本身不是終點——把這些方法整合成你每天實際執行的「個人作業系統」，才是真正的完成。

本單元帶你設計一份年度覆盤模板，以及把七大模組整合成一套可持續運作的個人管理系統的方法。同時，我們也討論如何讓這套系統保持「活著」——不是一次性的完美設計，而是每年持續迭代、適應你的人生變化。

// 本節重點

  • 年度覆盤的八個核心問題（生活、工作、健康、財務、關係、成長、貢獻、未竟之事）
  • 七大模組的整合關係圖：時間 → 生活 → 閱讀 → 工作 → 體重 → 理財 → 成長
  • 個人作業系統的每日、每週、每月、每季、每年維護節奏
  • 如何在系統崩潰後重建（不是從零開始，而是降速重啟）
  • 寫下給一年後自己的一封信
結業作業 完成你的「個人管理系統整合圖」：畫出這七個模組在你生活中的實際運作方式，並標記哪些已在執行、哪些還需要建立。最後，寫下一封寄給一年後自己的信，描述你希望那時候的生活是什麼樣子。 作業時間：2 小時
