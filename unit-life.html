生活管理
日常節律設計

生活管理不是讓你把每分鐘都填滿，而是設計一套可持續的日常骨架——讓好習慣自動發生，讓壞習慣難以出現。從晨間例程到環境設計，建立不靠意志力也能運作的生活系統。

核心觀念框架

Concept 01
節律設計

好的生活需要可預測的節奏。固定的起床、飲食與睡眠時間是一切的基礎。

Concept 02
習慣科學

習慣由提示、慣例、獎勵三部分組成。設計好迴路，行為就會自動發生。

Concept 03
環境力量

改變環境比改變意志力更有效。讓好的選擇變容易，讓壞的選擇變困難。

Concept 04
最小可行系統

系統要能在最差的狀況下仍可執行。複雜的計畫在壓力下最先崩潰。

課程單元

點擊展開內容
L01
生活現況盤點與理想日設計
Life Audit · 40 min
展開 ›

// 單元說明

改變生活的第一步，是如實看見現在的生活。許多人對自己的日常有美化過的印象——以為自己睡得夠、吃得健康、有在運動，但實際數字往往令人意外。

本單元透過「生活現況盤點表」，從睡眠、飲食、運動、社交、獨處、學習六個面向，記錄並評估你目前的生活品質基準線。接著，設計你的「理想日」——不是完美的一天，而是你真正想要且可持續的一天。

// 本節重點

  • 六大面向生活品質評分（睡眠 / 飲食 / 運動 / 社交 / 獨處 / 學習）
  • 找出生活中最讓你不滿意的三個領域
  • 設計「理想日時間表」——從起床到就寢
  • 理想日 vs. 現實日的差距分析
  • 設定本模組的個人改善目標
本節作業 完成生活現況盤點表，為六個面向各打一個分數（1–10）並寫下理由。接著花 30 分鐘設計你的理想日時間表，不需要完美，只需要誠實。 作業時間：1 小時
L02
晨間例程：掌控一天的開端
Morning Routine · 40 min
展開 ›

// 單元說明

一天的開始方式決定了這一天的基調。在你查看手機之前，在外界的需求湧入之前，晨間例程是你唯一能完全掌控的時段。許多高效能人士共同的特質，就是擁有一個穩定的晨間儀式。

本單元將帶你設計一套屬於自己的晨間例程——從 15 分鐘的極簡版本開始，逐步擴展到 60 分鐘的完整版本。重點不在於複製別人的晨間程序，而是找到真正讓你感到充電與清醒的活動組合。

// 本節重點

  • 為什麼晨間例程比夜貓習慣更容易建立
  • 晨間例程的四個核心元素：身體 / 心智 / 意圖設定 / 安靜時間
  • 設計三個版本：15 分鐘 / 30 分鐘 / 60 分鐘
  • 如何處理「不想起床」的阻力——睡眠與鬧鐘策略
  • 前一晚準備：讓晨間例程更容易執行的環境設定
本節作業 設計你的晨間例程三版本，並從本週開始執行 15 分鐘版本，連續七天。每天完成後簡短記錄感受（一句話即可），七天後評估哪些元素要保留、調整或刪除。 作業時間：7 天試行
L03
夜間例程：為明天做準備
Evening Routine · 35 min
展開 ›

// 單元說明

晨間例程決定了你如何開始一天，夜間例程則決定了你如何結束它——以及隔天早晨的品質。睡眠是人體最重要的恢復機制，而夜間例程的核心任務，就是幫助身心從工作模式切換到休息模式。

本單元聚焦於「數位宵禁」的設計、睡前準備的流程，以及如何透過隔日計畫讓大腦「下班」，真正從工作中抽離，進入深度睡眠。

// 本節重點

  • 睡眠品質對認知功能的直接影響
  • 數位宵禁：為什麼要在睡前 60 分鐘關螢幕
  • 夜間例程的三個核心任務：整理 / 放鬆 / 準備
  • 隔日計畫的快速執行（5 分鐘內完成）
  • 睡眠環境優化：溫度、光線、噪音控制
本節作業 設定一個「數位宵禁」鬧鐘，比預計就寢時間早 60 分鐘響起。這週開始在宵禁後只做：整理明日待辦、閱讀實體書、個人衛生三件事，並記錄睡眠品質變化。 作業時間：7 天試行
L04
習慣堆疊與觸發設計
Habit Stacking · 40 min
展開 ›

// 單元說明

習慣之所以難以建立，是因為我們試圖用意志力去驅動新行為，而不是借助既有習慣的慣性。習慣堆疊（Habit Stacking）是一種利用「現有習慣」作為觸發點，自然串接新習慣的方法。

公式很簡單：「在我做完 [現有習慣] 之後，我將做 [新習慣]。」這個方法的強大之處在於，它把新習慣嵌入已經自動化的行為序列中，大幅降低執行門檻。

// 本節重點

  • 習慣迴路：提示 → 慣例 → 獎勵 的運作機制
  • 習慣堆疊公式與實際應用範例
  • 識別你生活中已有的「錨點習慣」
  • 為新習慣設計最小可執行版本（2 分鐘法則）
  • 習慣追蹤：用打勾紀錄建立連勝動力
本節作業 列出三個你想建立的新習慣，為每個習慣找到一個「錨點習慣」，寫下完整的堆疊公式。從最小版本開始（每個新習慣不超過 2 分鐘），並用習慣追蹤表記錄執行狀況。 作業時間：30 分鐘設計 + 持續追蹤
L05
環境設計：讓空間幫你做決定
Environment Design · 45 min
展開 ›

// 單元說明

行為經濟學的研究一再證明：人的選擇在很大程度上受到環境的塑造，而非純粹的理性決策。我們以為自己有自由意志，但其實大多數時候，我們只是選了「最容易選的那個」。

環境設計的核心思想是：讓好的行為成為阻力最小的路徑，讓壞的行為增加摩擦力。不需要更強的意志力，只需要重新排列你的空間、工具與物品的位置。

// 本節重點

  • 摩擦力原理：降低好習慣的執行門檻
  • 視覺提示的力量：把想做的事放在眼睛看得到的地方
  • 工作空間設計：書桌、椅子、光線、雜物清理
  • 手機管理：放置位置、App 刪除、通知架構
  • 廚房與飲食環境：讓健康食物成為預設選項
本節作業 進行「環境審計」：走過你家和工作空間，找出三個讓你容易做壞選擇的地方（如手機放在床頭、垃圾食物放在桌上），以及三個可以讓好習慣更容易發生的環境調整。本週完成這六個改動。 作業時間：1 小時執行
L06
生活系統的維護與彈性
System Resilience · 40 min
展開 ›

// 單元說明

任何系統都會遇到干擾——出差、生病、情緒低潮、家庭突發事件。許多人在系統第一次被打亂後就完全放棄，這不是意志力不夠，而是系統缺乏彈性設計。

本單元的核心概念是「不要讓完美成為良好的敵人」。學習如何設計一個有「最低維持版本」的系統，讓你在最糟糕的情況下也能執行，同時建立「系統重啟」的機制，在偏離軌道後快速回來。

// 本節重點

  • 設計「緊急精簡版」例程：壞日子也能執行的最小版本
  • 不連續原則：錯過一天不算失敗，連續錯過才是問題
  • 系統重啟儀式：如何在中斷後快速回到軌道
  • 季節性審視：每季檢視並調整生活系統
  • 自我慈悲的重要性：放棄完美主義才能長期堅持
本節作業 為你的晨間例程和夜間例程各設計一個「緊急精簡版」（不超過 5 分鐘）。寫下在什麼情況下你會切換到精簡版，以及如何判斷自己已經準備好回到完整版。 作業時間：30 分鐘
