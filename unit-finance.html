理財管理
個人財務規劃

財務自由不是有錢人的專利，而是任何人只要早一天開始、持續執行正確方法就能逐步接近的目標。本模組從掌握現金流出發，建立記帳習慣、預算系統、緊急備用金，再到指數化投資的基礎概念，幫你打造一套可以自動運作的個人財務系統。

課程單元

點擊展開內容
L01
財務現況盤點與淨資產計算
Financial Audit · 45 min
展開 ›

// 單元說明

在制定任何理財計畫之前，你需要知道自己目前站在哪裡。大多數人對自己的財務狀況有一個模糊的感覺，但從未真正算清楚：我有多少資產？我有多少負債？我的淨資產是正數還是負數？

本單元帶你建立個人的「財務資產負債表」，列出所有資產（現金、存款、投資、不動產）與負債（信用卡未繳款、學貸、車貸、房貸），計算淨資產，並以此作為你理財旅程的起點座標。

// 本節重點

  • 資產分類：流動資產（現金、存款）/ 投資資產 / 實物資產
  • 負債分類：短期負債（信用卡）/ 長期負債（房貸、車貸）
  • 淨資產公式：資產總額 − 負債總額
  • 信用卡的真實成本：年利率 15–18% 的複利效應
  • 設定財務改善的三個階段里程碑
本節作業 用試算表或紙本，列出你所有的資產與負債，計算淨資產。這份清單不需要對任何人展示，只需要對自己誠實。寫下看到這個數字後的感受，以及你最想在六個月內改善的一項財務指標。 作業時間：1–2 小時
L02
記帳習慣：現金流的可視化
Expense Tracking · 40 min
展開 ›

// 單元說明

記帳不是為了限制自己，而是讓你對金錢的流向有意識。許多人在月底發現存款比預期少，卻說不清錢去了哪裡。記帳讓隱形的消費變得可見，是財務改善的第一步。

本單元的核心不是教你用什麼記帳工具，而是如何建立可持續的記帳習慣——從最低摩擦力的方法開始（如每天睡前 5 分鐘回顧），到養成習慣後再視需要加深記錄的精細度。重要的是開始，而不是完美。

// 本節重點

  • 支出五大類：食 / 住 / 行 / 育樂 / 其他
  • 記帳工具選擇：Spendee / 記帳城市 / YNAB / 紙本家計簿
  • 每月固定支出 vs. 變動支出的區分方式
  • 找出個人的「金錢漏洞」：最容易無意識花費的類別
  • 月結分析：收入、支出、儲蓄率三個關鍵數字
本節作業 選定一個記帳工具，從今天開始連續記帳 30 天（不需要更改消費行為，只是記錄）。月底統計支出各類別金額與佔比，計算本月儲蓄率（（收入−支出）/ 收入），找出金額最大的前三項非必要支出。 作業時間：每天 5 分鐘 × 30 天
L03
預算系統：50/30/20 法則
Budgeting · 40 min
展開 ›

// 單元說明

預算不是枷鎖，而是讓你「有許可地花錢」的工具。有了預算，你在花錢時不再有罪惡感，因為你知道這筆錢是計畫內的。50/30/20 法則是一套簡單、彈性的預算框架：50% 用於必要支出，30% 用於欲望，20% 用於儲蓄與投資。

本單元帶你根據個人的收入與生活狀況，建立一份個人化的預算分配表。同時討論如何處理「預算破表」的情況——不是從頭放棄，而是調整下個月的預算，讓預算成為可以持續迭代的動態系統。

// 本節重點

  • 50/30/20 預算法則的原理與彈性應用
  • 台灣高房價情境下的預算調整策略
  • 為每個支出類別設定月上限金額
  • 信封法（Envelope Method）：現金分裝或數位分帳戶
  • 每月預算結算：剩餘金額如何處理（結轉 or 轉入儲蓄）
本節作業 根據上個月的記帳結果，建立本月的預算分配表。將月收入按照 50/30/20 法則分配，再細分到各個支出類別，設定每個類別的月上限。把這份預算表貼在顯眼的地方，或存在手機記帳 App 中。 作業時間：1 小時
L04
緊急備用金與保險基礎
Financial Safety Net · 40 min
展開 ›

// 單元說明

在開始任何投資之前，有一件事比投資更重要：建立你的財務安全網。緊急備用金是存放在高流動性帳戶（如活存或高利活儲）的現金，專門用於面對失業、意外醫療、緊急修繕等突發狀況，讓你在危機來臨時不必被迫賣出投資或借貸。

建議的緊急備用金金額是 3–6 個月的月支出。在備用金建立完成之前，這應該是你的最高財務優先事項。本單元同時介紹基礎保險配置的思考框架——什麼保險是必要的，什麼是可選的。

// 本節重點

  • 緊急備用金目標金額：月支出 × 3（保守）到 × 6（穩健）
  • 備用金存放位置：高利活儲或貨幣型基金，不能放股市
  • 建立備用金的速度策略：將每月儲蓄的 50–80% 優先填滿備用金
  • 保險的核心邏輯：用小額保費轉移無法自行承擔的巨額風險
  • 台灣必備保險優先序：健保補充 → 意外險 → 定期壽險 → 重大疾病
本節作業 計算你的緊急備用金目標金額（月支出 × 4）。確認你目前的備用金存量，計算還需要多少，以及按照目前儲蓄速度需要幾個月達標。並開立一個獨立的高利活儲帳戶，專門作為緊急備用金帳戶，與日常消費帳戶分開。 作業時間：1 小時
L05
投資入門：複利與指數化投資
Index Investing · 50 min
展開 ›

// 單元說明

投資不是有錢人的專利，也不需要時刻盯盤。對於大多數人而言，最簡單、最有效的長期投資方式是：買進追蹤全市場的低成本指數基金，定期定額，長期持有，不要頻繁進出。

這個策略背後的邏輯是：幾十年的歷史數據顯示，超過 90% 的主動選股基金在長期無法打敗市場指數。與其花時間選股，不如用最低的費用買下整個市場，讓全球經濟增長為你工作。本單元帶你理解指數化投資的核心概念，並介紹台灣投資者可以使用的具體工具。

// 本節重點

  • 為什麼指數化投資優於主動選股（費用率 + 長期績效數據）
  • ETF（指數股票型基金）的運作原理與優點
  • 台灣投資者的核心工具：0050、00878、VT、VOO 簡介
  • 定期定額的心理優勢：平均成本法消除擇時焦慮
  • 投資前的心理準備：如何面對市場下跌而不恐慌賣出
本節作業 在緊急備用金建立完成的前提下，開立一個證券帳戶（若尚未有），選定一支你理解且認同的指數型 ETF，設定每月固定日期的定期定額扣款，金額從你感到舒適的數字開始（哪怕只是 1,000 元）。重要的是開始，而不是金額大小。 作業時間：2–3 小時開戶 + 設定
L06
自動化財務系統與長期規劃
Financial Autopilot · 40 min
展開 ›

// 單元說明

理財最大的敵人不是市場波動，而是人的情緒與惰性——忘記轉帳、被消費衝動帶走、在市場恐慌時賣出。自動化財務系統的目的，就是用制度替代意志力，讓正確的財務行為自動發生。

理想的自動化架構是：薪水入帳當天，自動扣款到緊急備用金帳戶、定期定額投資帳戶、短期儲蓄目標帳戶。剩下的才進入日常消費帳戶。這樣的設計讓你「花不到」本不該花的錢，而不需要每個月靠意志力轉帳。

// 本節重點

  • 多帳戶分流系統：收入帳戶 / 消費帳戶 / 儲蓄帳戶 / 投資帳戶
  • 薪水入帳後的自動轉帳設定（建議在薪資入帳後 1–2 天執行）
  • 短期儲蓄目標管理：旅行基金、換機基金、置裝基金的分設
  • 年度財務審視：每年 1 月重新評估投資比例與財務目標
  • 退休規劃入門：勞退自提 6% 的稅務優惠與長期效益
本節作業 設計你的個人「薪水分流系統」：畫出收入如何在入帳後自動分配到各帳戶的流程圖，設定自動轉帳指示。同時，確認你是否已開啟勞退自提，若尚未，評估是否啟用 6% 自提以獲得稅務優惠。 作業時間：2 小時設計 + 設定
