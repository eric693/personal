時間管理
工作分配優化

時間是唯一無法儲存的資源。這個模組從「時間感知」出發，帶你建立屬於自己的優先序系統，透過深度工作、時間區塊與精力管理，讓每一天從混亂走向清晰可控。

核心觀念框架

Concept 01
時間感知

先記錄，再優化。不了解時間流向，任何管理方法都是空談。

Concept 02
優先序

不是所有事情都重要。學會區分緊急與重要，把精力放對地方。

Concept 03
深度工作

保護你的專注時間。高難度的工作需要不被打斷的深度狀態。

Concept 04
系統思維

好的時間管理是一套可重複執行的系統，而非一次性的意志力爆發。

「忙碌不等於高效。真正有生產力的人，保護的是他們的注意力，而非他們的時間表。」
— Cal Newport, Deep Work

課程單元

點擊展開內容
L01
時間盤點：你的時間去哪了？
Time Audit · 45 min
展開 ›

// 單元說明

在改變任何習慣之前，你必須先知道自己的時間實際花在哪裡。大多數人對自己的時間使用有嚴重的誤判——以為花了兩小時在工作，實際上有效的深度時間可能不到四十分鐘。

本單元將帶你進行為期三天的「時間日誌」記錄，每半小時記錄一次自己在做什麼，並在結束後進行分類與分析。這是整個時間管理系統最重要的基礎步驟。

// 本節重點

  • 時間日誌的記錄方法與工具選擇
  • 活動分類：深度工作 / 淺層工作 / 維護性任務 / 浪費
  • 找出你的「時間漏洞」——最常不自覺消耗時間的情境
  • 計算你每天真正可用的「黃金工作時間」
  • 設定個人時間使用的基準線（baseline）
本節作業 連續三天，每 30 分鐘記錄一次自己的活動，使用紙本或 Notion 模板均可。完成後統計各類別佔比，找出前三大「時間消耗點」。 作業時間：3 天
L02
優先序矩陣：要事第一
Eisenhower Matrix · 40 min
展開 ›

// 單元說明

艾森豪矩陣將所有任務按照「緊急程度」與「重要程度」分為四個象限。大多數人的問題不是不夠努力，而是把大量時間花在「緊急但不重要」的事情上，忽略了真正能帶來長期改變的「重要但不緊急」任務。

本單元將帶你學會任務分類的思維，以及如何在每天開始前，用五分鐘確認自己的優先序，確保每天的精力花在最值得的事情上。

// 本節重點

  • 四象限定義：緊急重要 / 不緊急重要 / 緊急不重要 / 不緊急不重要
  • 為什麼「第二象限」是最被忽略、最重要的區域
  • 如何對「假緊急」任務說不
  • 每日三件要事（MIT）工作法
  • 建立個人任務清單的篩選機制
本節作業 列出目前待辦清單中的所有任務，用艾森豪矩陣進行分類。刪除或委派所有第四象限的任務，並為第二象限任務安排明確的時間。 作業時間：30 分鐘
L03
深度工作與淺層工作
Deep Work vs. Shallow Work · 45 min
展開 ›

// 單元說明

Cal Newport 在《深度工作力》中提出：在注意力高度分散的時代，能夠長時間專注的能力，將成為最稀缺也最有價值的技能。深度工作不僅讓你產出品質更高的成果，同時也帶來更強的成就感與心流體驗。

本單元將幫助你識別哪些是需要深度專注的工作，哪些是可以批次處理的淺層任務，並設計一套保護專注時間的環境與儀式。

// 本節重點

  • 深度工作的定義與衡量標準
  • 四種深度工作哲學：隱士型、雙峰型、節律型、新聞記者型
  • 設計你的「深度工作啟動儀式」
  • 消除分心來源：通知、環境、人際打斷
  • 淺層工作的批次處理策略
本節作業 本週安排兩次「深度工作時段」，每次 90 分鐘，關閉所有通知，記錄完成的成果與主觀感受。找出最適合自己的深度工作時段（早晨、下午或夜間）。 作業時間：1 週實驗
L04
時間區塊規劃法
Time Blocking · 40 min
展開 ›

// 單元說明

時間區塊規劃法是指在行事曆上為特定類型的工作預留固定的時間段，而非單純依賴待辦清單。當你的時間被提前「預訂」，就不容易被臨時的請求與分心打亂節奏。

本單元將帶你設計一份理想的「週計畫模板」，包含深度工作區塊、淺層工作區塊、休息與恢復時間，以及彈性緩衝時間，讓每週從一開始就有清晰的骨架。

// 本節重點

  • 時間區塊 vs. 待辦清單：兩者的互補關係
  • 設計你的理想週模板（Ideal Week Template）
  • 固定區塊 vs. 彈性區塊的比例配置
  • 如何處理臨時插入的任務不破壞計畫
  • 番茄鐘法作為區塊內的執行節拍器
本節作業 用行事曆或紙本，設計一份「理想週模板」，為深度工作、會議、郵件回覆、運動和個人恢復各安排固定時段，並在本週試行一次。 作業時間：1 小時設計 + 1 週試行
L05
精力管理與工作節律
Energy Management · 45 min
展開 ›

// 單元說明

管理時間的前提，是管理精力。人的注意力與認知資源在一天之中是有限且波動的——強迫自己在精力低谷時硬撐做高難度工作，不僅效率低落，還會加速耗盡。

本單元探討晝夜節律對工作能力的影響、如何識別個人的精力高峰時段，並設計「精力地圖」，讓對的任務配對對的精力狀態，最大化每一刻的工作品質。

// 本節重點

  • 晝夜節律與認知高峰時段的科學基礎
  • 找出你的「類型」：早鳥型 / 夜貓型 / 中間型
  • 繪製個人精力地圖（Energy Map）
  • 高精力 → 深度工作 ／ 中精力 → 創意工作 ／ 低精力 → 行政任務
  • 恢復策略：微休息、自然、運動與睡眠品質
本節作業 連續五天，在每天不同時段（上午、中午、下午、傍晚）記錄自己的專注度與精力分數（1–10）。畫出個人精力曲線，據此調整週模板中各類任務的時段安排。 作業時間：5 天觀察
L06
週覆盤與系統維護
Weekly Review · 40 min
展開 ›

// 單元說明

一套時間管理系統若沒有定期的維護與反思，很快就會崩解。週覆盤是讓系統持續運作的關鍵儀式——每週固定花三十分鐘，回顧過去一週的完成情況，處理遺留事項，並規劃下一週。

本單元將提供一份完整的「週覆盤問題清單」與操作流程，幫助你建立這個習慣，讓時間管理成為自動運作的背景系統，而非需要靠意志力維持的努力。

// 本節重點

  • 週覆盤的六個核心步驟（GTD 框架）
  • 清空收件匣：郵件、訊息、筆記、待辦
  • 回顧完成與未完成的任務，進行歸因分析
  • 確認下週目標與三大要事
  • 系統迭代：每月微調一次整體時間分配
本節作業 這週日，執行你的第一次完整週覆盤。使用本課提供的問題清單，花不超過 40 分鐘完成。並在行事曆上設定每週固定的覆盤時間（建議週日晚間或週一早晨）。 作業時間：每週 30–40 分鐘
